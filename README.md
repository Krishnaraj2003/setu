
## Description

Splitwise Clone API application for SETU interview. Please follow the below link for setup

## Installation


You would need nodejs. Install the latest [nodejs](https://nodejs.org/en/) version.

```bash
$ npm install
```
Install Redis - (Use docker) as it requires redis.
```bash
$ docker run -p 6379:6379 --name redis-redisjson redislabs/rejson:latest
```

## Running the app

```bash
$ npm run start
```
Use the postman collection attached for testing api.
